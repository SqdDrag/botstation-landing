Пользовательское соглашение

Дата последнего обновления: 14 июня 2026 г.

1. Термины

  • Сервис — платформа BotStation (botstation.xyz), предоставляющая инструменты для создания и хостинга Telegram-ботов по готовым шаблонам.
  • Пользователь — физическое лицо, использующее Сервис.
  • Бот — Telegram-бот, созданный и запущенный через Сервис.
  • Подписка — платная услуга, дающая право на хостинг и поддержку Ботов в течение оплаченного периода.
  • Триал — бесплатный пробный период (по умолчанию 12 часов) для нового Бота.

2. Предмет соглашения

Сервис предоставляет Пользователю доступ к платформе конструктора Telegram-ботов, включая:

  • Запуск и хостинг Ботов по готовым шаблонам (Магазин, Поддержка, Анкета, Платный канал).
  • Веб-панель управления Ботами.
  • Приём платежей за подписку.

Пользователь самостоятельно создаёт Telegram-ботов через @BotFather и предоставляет их токены для подключения к Сервису.

3. Регистрация и безопасность

  • Для использования Сервиса необходим аккаунт Telegram.
  • Токены Telegram-ботов хранятся в зашифрованном виде (Fernet/AES) и используются только для работы Ботов.
  • Пользователь несёт ответственность за сохранность токенов. Отзыв токена у @BotFather лишает Сервис доступа к Боту.
  • Пользователь не должен передавать токены третьим лицам.

4. Подписка и оплата

4.1. Триал

Каждый новый Бот получает бесплатный пробный период (по умолчанию 12 часов). По окончании триала Бот останавливается, данные сохраняются.

4.2. Подписка

  • Подписка оплачивается в USDT через Crypto Pay (@CryptoBot) или с внутреннего баланса.
  • Доступные периоды: 1, 3, 6, 12 месяцев. Скидки: 3 мес — 10%, 6 мес — 20%, 12 мес — 30%.
  • При наличии средств на балансе подписка продлевается автоматически.

4.3. Возврат средств

Возврат средств не предусмотрен. Пользователь может в любой момент отказаться от подписки, но ранее оплаченные периоды не компенсируются.

4.4. Баланс

  • Баланс пополняется через Crypto Pay в USDT.
  • Средства с баланса списываются за подписку Ботов и начисляются по реферальной программе.
  • Баланс не подлежит возврату в денежном виде.

5. Реферальная программа

  • Приглашая других пользователей по реферальной ссылке, Пользователь получает на баланс: 10% с оплат приглашённых первого уровня, 2% — второго уровня.
  • Начисления производятся с реальных платежей (Crypto Pay), не со списаний с баланса.
  • Сервис оставляет за собой право изменять условия реферальной программы с уведомлением пользователей.

6. Обязанности и ограничения

6.1. Пользователь обязуется:

  • Использовать Сервис в соответствии с законодательством своей страны.
  • Не использовать Боты для рассылки спама, мошенничества, распространения запрещённого контента.
  • Не пытаться получить несанкционированный доступ к Сервису или данным других пользователей.
  • Не нагружать Сервис чрезмерным количеством запросов (DDoS, абуз API).

6.2. Сервис не несёт ответственности за:

  • Временные сбои и недоступность Сервиса.
  • Действия Telegram (блокировка ботов, изменение API).
  • Данные клиентов Ботов — оператором этих данных является Пользователь.
  • Потерю данных при отзыве токена или удалении Бота.

7. Интеллектуальная собственность

Все права на платформу, шаблоны, код и дизайн принадлежат Сервису. Пользователь получает ограниченную лицензию на использование Сервиса в течение действия подписки.

8. Изменение условий

Сервис оставляет за собой право изменять настоящее соглашение. При существенных изменениях пользователи уведомляются через материнского бота. Продолжение использования Сервиса после изменений означает согласие с новыми условиями.

9. Прекращение использования

  • Пользователь может удалить все Боты и отказаться от Сервиса в любой момент.
  • Сервис может приостановить доступ при нарушении условий соглашения.
  • При удалении Бота его запись удаляется из реестра; файл базы данных может сохраняться для возможности восстановления.

10. Применимое право

Настоящее соглашение регулируется законодательством Российской Федерации. Все споры разрешаются путём переговоров; при недостижении согласия — в суде по месту нахождения Сервиса.
